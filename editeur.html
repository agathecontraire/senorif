Éditeur SENORIF
Ce que fait cette version :
• importe un ZIP complet du site existant ;
• reconstruit le plan avec les dossiers/sous-dossiers ;
• lit le contenu des fichiers HTML ;
• conserve les images/questionnaires du ZIP à l’export.
Tu peux importer le ZIP de ton dépôt GitHub. Si l’éditeur est hébergé dans le dépôt public GitHub Pages, le bouton “Charger dépôt GitHub” tente de lire directement les dossiers du dépôt.
